Guidelines and project structure
================================

`Project structure`_ gives a general idea of the tools and the workflow.
`Creating a project`_ describes how to create a project with the aforementioned
project structure. `Development and maintenance tasks`_ is a series of short
guides for common tasks such as contributing code and releasing a new version
of the software.
